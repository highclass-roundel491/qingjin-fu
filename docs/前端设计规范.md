# 青衿赋 - 前端设计规范

## 设计理念

青衿赋是一个沉浸式传统诗词学习与创作平台，设计风格应体现：
- 传统文化与现代技术的融合
- 优雅、诗意、沉浸式的用户体验
- 古典美学与现代简约的平衡

## 设计方向

**核心美学：现代东方 - 水墨诗意**
- 以中国传统水墨画为灵感
- 现代简约的布局与交互
- 诗意的动效与过渡
- 温润的色彩与留白

## 色彩系统

### 主色调
- 墨色：#2C3E50（深墨）、#34495E（中墨）
- 宣纸：#F8F9FA（浅）、#FDFEFE（白）
- 朱砂：#C0392B（强调色）
- 青绿：#16A085（辅助色）

### 渐变色
- 水墨渐变：从 #2C3E50 到 #95A5A6
- 晨曦渐变：从 #F39C12 到 #E74C3C
- 山水渐变：从 #16A085 到 #2ECC71

### 语义色
- 成功：#27AE60
- 警告：#F39C12
- 错误：#E74C3C
- 信息：#3498DB

## 字体系统

### 中文字体
- 标题：思源宋体 / Noto Serif SC（优雅、古典）
- 正文：思源黑体 / Noto Sans SC（清晰、现代）
- 诗词：方正楷体 / 华文楷体（传统、书法感）

### 英文字体
- 标题：Playfair Display（优雅衬线）
- 正文：Inter（现代无衬线）
- 代码：JetBrains Mono

### 字号规范
- 超大标题：48px / 3rem
- 大标题：36px / 2.25rem
- 中标题：24px / 1.5rem
- 小标题：20px / 1.25rem
- 正文：16px / 1rem
- 小字：14px / 0.875rem
- 注释：12px / 0.75rem

### 行高
- 标题：1.2
- 正文：1.6
- 诗词：1.8

## 布局系统

### 间距规范
- 基础单位：8px
- 常用间距：8px, 16px, 24px, 32px, 48px, 64px

### 容器宽度
- 最大宽度：1200px
- 内容宽度：960px
- 窄内容：720px

### 栅格系统
- 12列栅格
- 间距：24px
- 响应式断点：
  - 手机：< 768px
  - 平板：768px - 1024px
  - 桌面：> 1024px

## 组件设计

### 按钮
- 主按钮：朱砂色背景，白色文字
- 次按钮：墨色边框，墨色文字
- 文字按钮：无边框，墨色文字
- 圆角：8px
- 高度：40px（中）、36px（小）、48px（大）
- 悬停效果：轻微阴影 + 颜色加深

### 输入框
- 边框：1px 墨色边框
- 圆角：8px
- 高度：40px
- 聚焦：朱砂色边框 + 轻微阴影
- 占位符：浅灰色

### 卡片
- 背景：宣纸色
- 圆角：12px
- 阴影：0 2px 8px rgba(0,0,0,0.08)
- 悬停：阴影加深

### 导航栏
- 高度：64px
- 背景：半透明白色（毛玻璃效果）
- 阴影：0 2px 4px rgba(0,0,0,0.04)

## 动效设计

### 过渡时间
- 快速：150ms
- 标准：300ms
- 慢速：500ms

### 缓动函数
- 标准：cubic-bezier(0.4, 0.0, 0.2, 1)
- 进入：cubic-bezier(0.0, 0.0, 0.2, 1)
- 退出：cubic-bezier(0.4, 0.0, 1, 1)

### 动画效果
- 页面切换：淡入淡出 + 轻微位移
- 卡片悬停：上浮 + 阴影加深
- 按钮点击：轻微缩放
- 加载：水墨扩散效果
- 诗词展示：逐字显现（打字机效果）

## 图标系统

- 使用 Element Plus Icons
- 大小：16px（小）、20px（中）、24px（大）
- 颜色：继承文字颜色
- 风格：线性图标，简约现代

## 特色元素

### 水墨背景
- 使用 SVG 或 Canvas 绘制水墨纹理
- 半透明叠加
- 随机生成，每次访问不同

### 诗词卡片
- 竖排文字（传统排版）
- 印章装饰
- 宣纸质感背景
- 毛笔字体

### 装饰元素
- 中国结
- 祥云纹
- 回纹边框
- 竹简样式

## 响应式设计

### 移动端适配
- 字号适当缩小
- 间距紧凑
- 单列布局
- 底部导航栏

### 平板适配
- 两列布局
- 侧边栏可折叠

### 桌面端
- 三列布局
- 固定侧边栏
- 更多装饰元素

## 无障碍设计

- 颜色对比度 ≥ 4.5:1
- 可键盘操作
- 屏幕阅读器支持
- 焦点可见
- 语义化 HTML

## 性能优化

- 图片懒加载
- 字体子集化
- CSS 动画优先
- 避免重排重绘
- 使用 Web Fonts 加载策略

## 设计原则

1. **留白为美**：充分利用负空间，营造诗意氛围
2. **动静结合**：静态布局 + 动态交互
3. **虚实相生**：实体内容 + 装饰元素
4. **简约不简单**：简洁界面 + 精致细节
5. **传统与现代**：古典元素 + 现代交互

## 禁止事项

- ❌ 使用过于花哨的动画
- ❌ 过度使用装饰元素
- ❌ 颜色过于鲜艳刺眼
- ❌ 字体过小难以阅读
- ❌ 布局过于拥挤
- ❌ 使用低质量图片
- ❌ 忽视移动端体验
